# 实验五：自定义分形实验

## 实验目的
- 理解分形的自相似性和生成机制
- 掌握分形生成算法的编程实现
- 探索自定义分形结构的数学与美学特征

## 实验内容
1. 设计并实现一种自定义分形（如分形树、龙曲线、Koch雪花等），可自由选择分形类型。
2. 编写Python程序生成该分形的点集或图像。
3. 可选：尝试分析该分形的分形维数。
4. 绘制分形图像并进行美学探索。
